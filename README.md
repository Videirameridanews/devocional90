## 🚀 Cómo publicar en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En **Source**, selecciona `main` branch y carpeta `/ (root)`
4. Haz clic en **Save**
5. En 1-2 minutos tu sitio estará en vivo ✅
